# A股每日选股雷达（astock-radar）

零成本自动化选股研究管线：

```
GitHub Actions（每交易日17:30自动运行）
  └─ fetch_data.py：akshare 免费拉取全市场行情/估值/业绩预告/快报/报表/行业板块
       └─ 提交到本仓库 data/latest/
            └─ Claude 定时任务（18:00）：clone 本仓库 → analyze.py 多因子评分
                 → 新闻/公告搜索验证 → 生成 HTML 日报推送
```

## 评分框架

- **质量 30%**：ROE、毛利率、经营现金流兑现度
- **估值 25%**：行业内 PE/PB 分位（越便宜越高分）
- **业绩催化 25%**：业绩预告（预增/扭亏及幅度）、业绩快报净利同比
- **左侧位置 20%**：60日涨跌幅处于横盘/回调区间加分，已大涨/换手过热扣分

评级：G1（顶级候选，硬催化+左侧+无红旗，宁缺毋滥）/ G2（强候选）/ G3（观察池）。
红旗：疑似价值陷阱、净利大跌、换手过热、当日大涨、60日涨幅已高。

## 手动触发

仓库页面 → Actions → fetch-astock-data → Run workflow（`force=1` 可在非交易日强制运行）。
